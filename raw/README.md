# Raw Sources

This directory stores immutable source material for the wiki.

## Subdirectories

- `raw/inbox/` - new files waiting to be normalized or ingested.
- `raw/papers/` - stable local paper store.
- `raw/assets/` - downloaded images, figures, and attachments referenced by notes.

Do not edit source content in place. The wiki should absorb and synthesize sources into `wiki/`.
